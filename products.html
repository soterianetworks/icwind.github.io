产品介绍

  • 物联网数据平台

    IOT DATA STREAMING

    芯翼达的物联网数据平台支持多种设备数据传输模式，包括 MQTT， TCP， HTTP， UDP等； 采集的数据在设备网关经过数据的标准化，通过 ESB 由微服务平行处理， 从大量数据中采集商业信息，支持数据可视化和 API 数据共享

  • 物联网安全芯片

    KEYIN TECHNOLOGY

    物联网安全是云安全技术的扩展，其独特性体现在设备端的安全；我们开发的 KEYIN 安全芯片支持不可复制的 密钥，可用于数据加密，防伪

  • 云集成平台

    CLOUD INTEGRATION

    关键词：多租户，信息资产（人，设备，APP， API），管理，集成，安全


    特点： 与云服务商提供的服务相比较，这样的系统更加灵活，安全，为企业提供技术的独立性


    功能模块：API 网关，SAML 单点登录，OAUTH2 认证，MFA 多因子验证，权限管理，审计

  • 系统监控平台

    SYSTEM MONITORING

    芯翼达的监控产品提供实时状态与时间段趋势的分析，支持计算机系统的核心指标，包括 CPU， MEMORY， DISK；支持网络状态拓扑图； 还有针对应用层面的功能，包括服务器，数据库
