Products Line

Chip Design

Physical Unclonable Functionality

PUF is the “silicon biometrics” that exploit the intrinsic and ubiquitous variations in semiconductor integrated circuits (IC) to generate random but unique, unpredictable while reproducible, and most importantly, unclonable hardware fingerprints.

Compressive Sensing

The compressive sensing technology by Soteria Networks is a perfect solution that solves both the bandwidth and security issue. It compresses and transforms the data in a unitary step. Even better, this is done at the chip level, much faster than the software method. The transformed data can transit through even public wireless network without worrying about man in the middle attack.

API Gateway

The API Gateway provides a great infrastructure component that simplifies service oriented architect implementation. APIs are registered with the API gateway. Requests for the services are routed through the API gateway. The API gateway also implements the authorization enforcement based on the privileges for the principal in the request.

OAuth2 Authorization Server

OAuth2 provides authorization flows for web applications, desktop applications, mobile phones, and living room devices. It offers similar single sign on capability, but is a better choice when it comes to API level authentication for server to server communication. It handles the authorization by a resource owner when a client accesses a resource server protected by an OAuth2 authorization server.

Solutions

Data Pool

Enterprise companies typically have many data storage systems. It’s very critical for business to leverage the rich data content to harvest more revenue. This is where data pool or data lake concept comes into picture. A business will build micro services to expose data access APIs. As the number of APIs grows, how to manage and control the access to the APIs becomes a prominent issue. The cloud integration platform has a build-in API gateway through which a business can register/expose service APIs. The gateway is an authorization/policy enforcement point where API access is checked against the privileges associated with the principal in the request.

Business Merge & Acquisition

Business merge and acquisition often result in information islands. Quickly connecting the systems together not only reduces business overhead, but also creates new revenue opportunities. With Soteria Networks’ cloud integration platform, existing systems can participate in SSO federation using SAML or OAuth2, the best standards currently used by the industry. Users can access additional applications based on privilege settings.

Cloud Computing

A new cloud data center should not settle with providing hardware resources to its customers, Value added services, such as API gateway and Identity & Access Management, are highly desirable for additional revenue. The cloud integration platform by Soteria Networks is a perfect match to quickly enable those features.
